# MVC_Project_Lite

Estructura de carpetas base para uso en un proyecto web pequeño que no necesite demasiado orden para ser legible.

```
├───lib
│   └───demo-lib
├───php
├───public
│   ├───src
│   │   ├───css
│   │   ├───fonts
│   │   │   └───demo-fonts
│   │   ├───images
│   │   └───js
│   └───views
└───test
```
